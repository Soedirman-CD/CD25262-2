# Dual Axis Solar Panel Tracker
  Dual Axis Solar Panel Tracker merupakan sistem energi terbarukan yang dirancang untuk memaksimalkan penyerapan energi matahari dengan cara menggerakkan panel surya secara otomatis mengikuti arah pergerakan matahari. Berbeda dengan panel surya statis, sistem ini menggunakan dua sumbu pergerakan (horizontal dan vertikal) sehingga panel dapat selalu berada pada posisi optimal terhadap cahaya matahari sepanjang hari.
  Proyek ini mengintegrasikan sensor, mikrokontroler, dan mekanisme aktuator untuk meningkatkan efisiensi panel surya serta mendukung pengembangan teknologi energi berkelanjutan.

# Tujuan Proyek
1. Merancang dan mengimplementasikan sistem Solar Panel Tracking dual axis berbasis mikrokontroler untuk meningkatkan efisiensi penyerapan energi surya dibandingkan panel statis.
2. Membangun sistem penyimpanan energi berbasis baterai sebagai catu daya mandiri (off-grid) guna menjaga operasional router dan server WiFi saat terjadi pemadaman listrik dari PT PLN (Persero).
3. Mengintegrasikan sistem monitoring berbasis Internet of Things (IoT) untuk memantau parameter kelistrikan seperti tegangan, arus, daya, dan kapasitas baterai secara real-time.

# Manfaat Proyek
1. Meningkatkan keandalan operasional server dan jaringan WiFi dengan menyediakan sumber daya cadangan berbasis energi terbarukan.
2. Meningkatkan efisiensi pembangkitan energi surya melalui penerapan sistem solar tracking dua sumbu.
3. Memberikan kemudahan pemantauan kondisi sistem secara jarak jauh melalui teknologi IoT sehingga potensi gangguan dapat dideteksi lebih dini.

#  Komponen yang Digunakan
1. Mikrokontroler ESP32
2. Sensor LDR, Sensor Tegangan, dan Sensor Arus
3. Servo Motor
4. Solar Panel

# Fitur Utama
1. Monitoring posisi panel surya secara real-time
2. Monitoring intensitas cahaya matahari
3. Monitoring pergerakan dual axis (horizontal & vertikal)
4. Sistem berbasis mikrokontroler
5. Tampilan data real-time pada dashboard
